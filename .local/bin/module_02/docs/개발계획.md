# 슬라임의 역습 개발 계획서

## 프로젝트 개요

**프로젝트 명**: 슬라임의 역습 (Slime Rising)
**장르**: 브라우저 기반 액션 게임
**개발 언어**: HTML5, CSS3, JavaScript (Vanilla)
**타겟**: 웹 브라우저 (데스크톱)

## 게임 컨셉

플레이어는 슬라임을 조작하여 다가오는 영웅들을 흡수하고 그들의 능력을 획득합니다. 5가지 영웅 타입을 모두 흡수하면 승리합니다.

## 현재 구현 상태

### 완료된 기능

1. **플레이어 시스템**
   - 슬라임 캐릭터 이동 (WASD / 방향키)
   - HP, 공격력, 방어력 스탯 시스템
   - 경험치 및 레벨업 시스템
   - 능력 흡수 메커니즘

2. **적 시스템**
   - 5가지 영웅 타입 (전사, 마법사, 사제, 궁수, 도적)
   - 각 타입별 고유 행동 패턴
   - 근접/원거리 공격 시스템
   - 웨이브 기반 난이도 증가

3. **전투 시스템**
   - 실시간 전투
   - 투사체 시스템 (마법사, 궁수)
   - 충돌 감지 및 데미지 계산
   - 치유 메커니즘 (사제)

4. **UI/UX**
   - 타이틀 화면
   - 게임 HUD (HP, 레벨, 웨이브 정보)
   - 능력 아이콘 표시
   - 게임 오버 / 승리 화면

5. **비주얼**
   - CSS 기반 캐릭터 렌더링
   - 애니메이션 효과 (바운스, 걷기, 흡수)
   - 데미지 표시 효과
   - 레벨업 이펙트

## 기술 스택 검토

### 현재 사용 중
- Vanilla JavaScript (ES6+)
- HTML5 Canvas (미사용, DOM 기반)
- CSS3 Animations
- RequestAnimationFrame

### 고려 사항
- **Canvas API**: 더 많은 엔터티 렌더링 시 성능 개선
- **Web Audio API**: 사운드 시스템 구현 시 필요
- **TypeScript**: 타입 안정성 및 유지보수성 향상
- **번들러**: Webpack/Vite 도입으로 모듈 관리 개선
